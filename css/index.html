Tutorial for Csongor (CSS)

CSS steht für "Cascading Style Sheets" und ist für den Design der Website.

Wie benutzt man CSS?

Inline:

Man benutzt beim Inline Modi denn Parameter "style". Dies funktioniert in jedem visiblen Objekt. Das benutzt man meistens wenn man die CSS-Deklaration nur einmal braucht.

Beispiel:

Code Kopieren 1 <p style="...">Hello World</p>

Internal:

Im <head> Element kann man den <style> Element benutzten und dort direct CSS schreiben. Das benutzt man meist wenn es mehrere Objecte mit dem gleichen CSS-Deklarationen gibt.

Beispiel:

Code Kopieren 1 <head>
2 <style>...</style>
3 </head>

External:

Im <head> Element kann man den <link rel="stylesheet" type="text/css" href="..."/> Element benutzten wo man im "href" der Ort der CSS Datei definiert und.

Beispiel:

Code Kopieren 1 <link rel="stylesheet" type="text/css" href="../src/fonts/monaco.css"/>

CSS-Selektoren

CSS kann Objekte formattieren, aber dafür muss man Elemente wenn CSS Internal/External benutzt erst mal auswählen.

Typ-Selektor:

Wenn man die formattierung von allen Elemente des den gleichen Typen (<h1>, <p>, <hr>) ändern willst benutzt du den Typen-Selektor

Beispiel:

Code Kopieren 1 h1 {
2 ...
3 }

Klassen-Selektor:

Wenn man die formattierung von allen Elementen mit der gleichen Klasse (die Klasse setzt man mit dem "class" Parameter) ändern willst benutzt du den Klassen-Selektor

Beispiel:

Code Kopieren 1 .highlighted {
2 ...
3 }

ID-Selektor:

Wenn man die formattierung von von einem Element mit bestimmter ID (die ID setzt man mit dem "id" Parameter) ändern willst benutzt du den ID-Selektor

Beispiel:

Code Kopieren 1 #download-button {
2 ...
3 }

Universalselektor:

Wenn man die formattierung von allen Elementen in der Website ändern willst benutzt du den Universalselektor

Beispiel:

Code Kopieren 1 * {
2 ...
3 }

<img> Images

<img src="" alt=""> also eine Image (Bild). Es macht ein Bild mit dem Path/URL in "href". <img> hat auch das "alt" Parameter dass eine kurze Beschreibung über dein Bild gibt falls dass Bild nicht ladet.

Tipp: Das "width" Parameter setzt die horizontale Grösse des Bilds.

Beispiel:

Code Kopieren 1 <img src="https://example.com/mountainscape.jpeg" alt="Mountains" width="400px"><br>
2 <img src="https://example.com/image.png" alt="Example Image"" width="200px">

<a> Anchor

<a href=""> auch als "Anchor" bekannt ist das Element für Hyperlinks. Es macht ein Element das dich zu einem Path oder URL führt.

Anchors sind nicht nur zu Text limitiert, zum Beispiel würde "<a href=""><img ...></img></a>" auch funktionieren.

Beispiel:

Code Kopieren 1 <a href="https://youtube.com/">Zu YouTube</a><br>
2 <a href="https://google.com/">
3 <img src="https://google.com/logo.png" alt="Google logo">
4 </a>
Zu YouTube

<ul> oder <ol> oder <dl> Lists

<ul> (Unordered list):

<ul> oder "Unordered list" ist eine Liste die Punktiert ist (Mit •).
<il> oder "List item" ist das Element das für jeden Punkt im "ul"" benutzt wird.

Beispiel:

Code Kopieren 1 <p>Einkaufsliste:</p><br>
2 <ul>
3 <li>Schorle</li>
4 <li>Appenzeller</li>
5 </ul>

Einkaufsliste:

  • Schorle
  • Appenzeller

<ol> (Ordered list):

<ol> oder "Ordered list" ist eine Liste die die Numeriert wird (1, 2, 3...).
<il> oder "List item" ist das Element das für jeden Punkt im "ol"" benutzt wird.

Beispiel:

Code Kopieren 1 <p>How to HTML:</p><br>
2 <ol>
3 <li>Make Body</li>
4 <li>Improvise</li>
5 </ol>

How to HTML:

  1. Make Body
  2. Improvise

<dl> (Description list):

<dl> oder "Description list" ist eine Liste die ein Term beschreibt.
<dt> oder "Description term" ist das Element das für den Namen des Terms.
<dd> oder "Description description" ist das Element das für die Beschreibung des Termes.

Beispiel:

Code Kopieren 1 <p>HTML Basics:</p><br>
2 <dl>
3 <dt>h1</dt>
4 <dd>Heading</dd>
5 <dt>p</dt>
6 <dd>Paragraph</dd>
7 <dt>a</dt>
8 <dd>Anchor</dd>
9 </dl>

<span> oder <div> Container

<span>:

<span> ist ein Element das nur für CSS benutzt wird und ändert in sich selber nichts.

Beispiel:

Code Kopieren 1 Hello <span style="...">World</span>
Hello World

<div> (Division):

<div> oder "Division" ist wie ein <span> aber macht seinen eigener Platzt für den Inhalt.

Beispiel:

Code Kopieren 1 Hello <div>World</div>
Hello
World

<b> oder <i> Formatter

<b> (Bold):

<b> oder "Bold" ist das Element das alles im Element fett macht.

Beispiel:

Code Kopieren 1 Hello <b>World</b>
Hello World

<i> (Italic):

<i> oder "Italic" ist das Element das alles im Element kursiv macht.

Beispiel:

Code Kopieren 1 Hello <i>World</i>
Hello World
