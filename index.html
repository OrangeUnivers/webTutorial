Tutorial for Csongor (HTML)

HTML steht für "Hypertext Markup Language" und ist für die Strukterierung der Website.

Einleitung

Schreib ! und drück dann TAB um die normale HTML Strukturierung zu erstellen.

Der folgende Code wird dabei kreiert:

Code Kopieren 1 <!DOCTYPE html>

5 <meta name="viewport" content="width=device-width, initial-scale=1.0">
6 <title></title>
7 </head>
8 <body>
9
10 </body>
11 </html>

<h1> bis <h6> Heading

<h1-6> auch als "Heading" bekannt ist das Element für Titel/Untertitel. Benutze es um Kapitel oder Unterteilungen auf deiner Website zu zeigen.

Achtung: <h1> sollte nur einmal auf einer Seite sein weil Suchmaschinen sie zur formatierung benutzten.

Beispiel:

Code Kopieren 1 <h1>Heading 1</h1>
2 <h2>Heading 2</h2>
3 <h3>Heading 3</h3>
4 <h4>Heading 4</h4>
5 <h5>Heading 5</h5>
6 <h6>Heading 6</h6>

<p> Paragraph

<p> auch als "Paragraph" bekannt ist das Element für Text. Benutze es für Textbereiche.

Tipp: <br> stellt den nächsten Element auf der nächsten Linie.

Beispiel:

Code Kopieren 1 <p>Willkommen zu meiner Website</p>

Willkommen zu meiner Website

<img> Images

<img src="" alt=""> also eine Image (Bild). Es macht ein Bild mit dem Path/URL in "href". <img> hat auch das "alt" Parameter dass eine kurze Beschreibung über dein Bild gibt falls dass Bild nicht ladet.

Tipp: Das "width" Parameter setzt die horizontale Grösse des Bilds.

Beispiel:

Code Kopieren 1 <img src="https://example.com/mountainscape.jpeg" alt="Mountains" width="400px"><br>
2 <img src="https://example.com/image.png" alt="Example Image" width="200px">

<a> Anchor

<a href=""> auch als "Anchor" bekannt ist das Element für Hyperlinks. Es macht ein Element das dich zu einem Path oder URL führt.

Anchors sind nicht nur zu Text limitiert, zum Beispiel würde "<a href=""><img ...></img></a>" auch funktionieren.

Beispiel:

Code Kopieren 1 <a href="https://youtube.com/">Zu YouTube</a><br>
2 <a href="https://google.com/">
3 <img src="https://google.com/logo.png" alt="Google logo">
4 </a>
Zu YouTube

<ul> oder <ol> oder <dl> Lists

<ul> (Unordered list):

<ul> oder "Unordered list" ist eine Liste die Punktiert ist (Mit •).
<il> oder "List item" ist das Element das für jeden Punkt im "ul"" benutzt wird.

Beispiel:

Code Kopieren 1 <p>Einkaufsliste:</p><br>
2 <ul>
3 <li>Schorle</li>
4 <li>Appenzeller</li>
5 </ul>

Einkaufsliste:

  • Schorle
  • Appenzeller

<ol> (Ordered list):

<ol> oder "Ordered list" ist eine Liste die die Numeriert wird (1, 2, 3...).
<il> oder "List item" ist das Element das für jeden Punkt im "ol"" benutzt wird.

Beispiel:

Code Kopieren 1 <p>How to HTML:</p><br>
2 <ol>
3 <li>Make Body</li>
4 <li>Improvise</li>
5 </ol>

How to HTML:

  1. Make Body
  2. Improvise

<dl> (Description list):

<dl> oder "Description list" ist eine Liste die ein Term beschreibt.
<dt> oder "Description term" ist das Element das für den Namen des Terms.
<dd> oder "Description description" ist das Element das für die Beschreibung des Termes.

Beispiel:

Code Kopieren 1 <p>HTML Basics:</p><br>
2 <dl>
3 <dt>h1</dt>
4 <dd>Heading</dd>
5 <dt>p</dt>
6 <dd>Paragraph</dd>
7 <dt>a</dt>
8 <dd>Anchor</dd>
9 </dl>

<table> Table

<table> ist das Element der Tabele
<tr> oder "Table row" ist das Element für jeder Reihe der Tabele.

<span> oder <div> Container

<span>:

<span> ist ein Element das nur für CSS benutzt wird und ändert in sich selber nichts.

Beispiel:

Code Kopieren 1 Hello <span style="...">World</span>
Hello World

<div> (Division):

<div> oder "Division" ist wie ein <span> aber macht seinen eigener Platzt für den Inhalt.

Beispiel:

Code Kopieren 1 Hello <div>World</div>
Hello
World

<b> oder <i> Formatter

<b> (Bold):

<b> oder "Bold" ist das Element das alles im Element fett macht.

Beispiel:

Code Kopieren 1 Hello <b>World</b>
Hello World

<i> (Italic):

<i> oder "Italic" ist das Element das alles im Element kursiv macht.

Beispiel:

Code Kopieren 1 Hello <i>World</i>
Hello World

Csongor Special! (Stream auf Website)

<iframe> (Inline frame):

<iframe> oder "Inline frame" ist ein Element das eine eigene HTML Seite laden kann. Um auf der Webseite zu streamen, braucht man einen <iframe> mit https://www.youtube.com/embed/YOUR_STREAM_KEY (YOUR_STREAM_KEY ist dein eigener Stream Key).

Tipp: Um dein Stream nur auf deiner Website verfügbar zu machen kann man einfach den Stream als nicht gelistet einrichten.

Beispiel:

Code Kopieren 1 <iframe width="560" height="315" src="https://www.youtube.com/embed/YOUR_STREAM_KEY" frameborder="0" allowfullscreen></iframe>
